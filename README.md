# 📋 Sistema de Cadastro de Funcionários

https://crud-st.streamlit.app/

Aplicação web desenvolvida com **Python** e **Streamlit** para gerenciamento de funcionários, com funcionalidades de cadastro, consulta, edição, exclusão e exportação de dados.

---

## 🚀 Funcionalidades

- **Cadastro** de novos funcionários com nome, cargo, salário, cidade e estado
- **Consulta** de funcionários por índice
- **Edição** dos dados cadastrados
- **Exclusão** de registros com confirmação de segurança
- **Exportação** da base de dados em formato `.csv` compatível com Excel

---

## 🛠️ Tecnologias utilizadas

- [Python 3.x](https://www.python.org/)
- [Streamlit](https://streamlit.io/)
- [Pandas](https://pandas.pydata.org/)
